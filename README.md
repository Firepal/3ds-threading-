# 3ds-threading++
Let's try to implement C++ standard library threads using libctru for the sake of easier porting.

This project plans to implement the following libraries:
- `<thread>`
- `<mutex>`
- `<shared_mutex>`
- `<condition_variable>`

# Usage
_Note: This project is not readily functional yet._
<ol>
<li>Include the <code>libctru</code> library in your compiler settings.</li>
<li>Include the files inside <code>include_src</code> in your C++ project.</li>
<li>Profit!</li>
</ol>
